--JBurst--

A library made for Java Swing modeled from HaxeFlixel that adds sprites with animation and simple transformation methods.

I always found it frustrating how JLabel only really supported static images.
(This was once intended to be a very basic game engine, like HaxeFlixel, but there wasn't really much purpose in copying them)

--What it does--

The main thing this library does is add animated images.

This is accomplished through the use of spritesheets and frame atlases.
Supported spritesheet formats include ones with even spacing (can be cut into regions of identical size), sparrow texture atlases, and json exture-packer atlases.

On top of animation, it also makes transformations such as rotation and dilation (growth) much, much simpler.
When transforming graphics, offsets are automatically calculated and applied to the sprite to keep it in the position its intended to be in.
